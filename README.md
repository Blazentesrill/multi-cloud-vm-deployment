# Multi-Cloud VM Deployment & Security Project

## Overview
Deployed virtual machines across multiple cloud providers and hosted a live webpage using Apache.

## Cloud Providers Used
- Microsoft Azure
- Google Cloud Platform (GCP)
- Oracle Cloud
- IBM Cloud

## Steps Performed
1. Created VM instances
2. Configured firewall rules (HTTP/SSH)
3. Installed Apache web server
4. Hosted custom webpage
5. Verified public access
6. Terminated resources to avoid costs
